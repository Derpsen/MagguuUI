# Changelog

All notable changes to MagguuUI will be documented in this file.

## [1.0.0] - 2026-02-08

### Added
- Initial public release
- Guided installer with step-by-step profile installation
- One-click "Install All" to apply all profiles at once
- Sequential profile loading for new characters with automatic detection
- WowUp addon import string with in-game copy popup
- Profile support for ElvUI (profile, private, global, aura filters)
- Profile support for Plater, BigWigs, Details, BetterCooldownManager
- Profile support for Blizzard EditMode layouts
- Minimap button with drag-to-reposition
- Addon Compartment integration
- Settings panel with per-addon profile management
- Overwrite confirmation for existing profiles
- Addon enabled/disabled detection per installer step
- Chat commands: `/mui install`, `/mui settings`, `/mui minimap`, `/mui version`, `/mui status`
- Custom MagguuUI branding (blue/silver theme)
